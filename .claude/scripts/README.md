# One-Time Scripts

**Purpose:** Temporary scripts for validation, diagnostics, and one-off tasks.

**Rules:**
1. Scripts here are NOT part of the application
2. Name with purpose + date: `validate_rls_20251203.py`
3. DELETE after use (unless needed for documentation)
4. Never import these in production code

**When to use:**
- Database validation scripts
- One-time data migrations
- Diagnostic tools
- Testing utilities

**Cleanup:**
Delete scripts older than 7 days unless explicitly documented as reference material.
