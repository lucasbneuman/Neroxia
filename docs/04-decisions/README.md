# Decisions

Este directorio queda reservado para ADRs y decisiones estables que formalicen excepciones, cambios estructurales o politicas duraderas.

## Cuando agregar una ADR

- cuando una decision no entra naturalmente en `SRS`, `SAD` o `SDD`
- cuando se necesita registrar una excepcion formal a la jerarquia documental
- cuando un cambio arquitectonico estable merece trazabilidad propia

Si no hay ADRs todavia, el directorio puede permanecer con este indice.
